## Usage
### Local developement
```
npm start
```
### Deploy to Cloud Foundry
Please make sure you installed the [Cloud Foundry CLI](https://github.com/cloudfoundry/cli) cli and your are signed in ``cf login``
```
npm run deploy:cf
```
